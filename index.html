AI INTEGRATED IOT BASED SMART FAN SYSTEM

FOR ADAPTIVE CONTROL AND SAFETY MONITORING

An intelligent IoT-based smart fan system designed for automatic temperature control, motion detection, energy efficiency, and smart safety monitoring.

Project Overview

The Smart Fan System is an AI integrated IoT project that automatically controls fan operation based on environmental conditions and user requirements.

Key Features

  • Four intelligent operating modes
  • Temperature based fan speed control
  • Motion detection using ESP32-CAM
  • Servo radar scanning system
  • Blynk IoT application control
  • Telegram alert system

Working Modes

1. Manual Mode

Default operating mode. Relay1 controls the fan and Relay5 controls rotation. The fan runs continuously until the user changes the mode.

2. Semi Auto Mode

The user manually selects the fan speed through the Blynk application. Relay2 controls low speed, Relay3 medium speed, and Relay4 high speed.

3. Auto Mode

Uses DHT22 sensor, ultrasonic sensor, ESP32-CAM motion detection, and servo scanning. Fan speed changes automatically according to room temperature.

4. Spy Mode

All relays and OLED display turn OFF while motion detection remains active. A Telegram alert is sent when motion is detected.

Temperature Based Control

Below 28°C

Low Speed

28°C to 35°C

Medium Speed

Above 35°C

High Speed

Components Used

  • ESP32 Microcontroller
  • ESP32-CAM Module
  • DHT22 Temperature Sensor
  • Ultrasonic Sensor
  • Servo Motor
  • Relay Modules
  • OLED Display

User Instructions

  • Ensure the socket is adaptable for the fan.
  • Maintain stable network and power supply.
  • Do not share Wi‑Fi credentials.
  • Do not share Blynk login details.
  • System starts in Manual Mode.
  • Turn OFF switches before changing modes.
  • Ensure rotation is OFF before changing modes.
